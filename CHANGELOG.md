## 1.1.0

#### New Features

- Full rewrite of the main script. All theme settings are available from the Command Panel now.
- Support for the Terminal package. New option to use more traditional colors in terminal.
- More visible matching brackets highlighting.

## 1.0.0

#### New Features

- Switched to Base2Tone syntax theme.
- Include 20 Base2Tone color schemes in 4 available styles.
- Preview of each selected theme on all open files while browsing through the list of available themes.
